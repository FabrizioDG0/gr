# Roman Pizza Site 🍕

A full-stack web app that teaches how to make Roman-style pizza! Built with React (frontend) and Flask (backend).

## 🔧 Tech Stack
- React (client/)
- Flask (server/)

## 🚀 How to Run

### Frontend (React)
```bash
cd client
npm install
npm run build
```

### Backend (Flask)
```bash
cd server
pip install -r ../requirements.txt
python server.py
```

Visit: http://localhost:5000
